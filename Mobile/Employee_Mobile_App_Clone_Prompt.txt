# Wrap N' Track Employee Mobile App Clone - Comprehensive Development Prompt

## Project Overview
Create a comprehensive React Native mobile application that clones the employee side of the Wrap N' Track website, providing identical functionality, data structures, and user experience optimized for mobile devices.

## Current Website Analysis

### Employee Side Features Identified:
1. **Dashboard** - Analytics overview with inventory, sales, and order metrics
2. **Inventory Management** - Complete product management with CRUD operations
3. **Order Management** - Order tracking, status updates, and customer communication
4. **Customer Management** - Customer database with search, filter, and management
5. **Supplier Management** - Supplier database and relationship management
6. **Reports System** - Sales reports, inventory reports, and analytics
7. **Account Management** - User profile and role-based access control

## Technical Architecture

### Core Technologies
- **Framework**: React Native with Expo
- **Navigation**: React Navigation (Drawer, Stack, Tab Navigators)
- **State Management**: Context API with useReducer
- **UI Components**: React Native Paper + Custom Components
- **API Integration**: Axios with interceptors
- **Authentication**: JWT-based with role-based access control
- **Database**: PostgreSQL (same as website)
- **Backend**: Node.js/Express (reuse existing API)

## API Integration

### Base Configuration
```javascript
const api = axios.create({
  baseURL: 'http://your-server.com/api',
  timeout: 15000,
  headers: {
    'Content-Type': 'application/json'
  }
});

// Request interceptor for auth
api.interceptors.request.use(async (config) => {
  const token = await AsyncStorage.getItem('authToken');
  if (token) {
    config.headers.Authorization = `Bearer ${token}`;
  }
  return config;
});

// Response interceptor for error handling
api.interceptors.response.use(
  (response) => response,
  async (error) => {
    if (error.response?.status === 401) {
      await AsyncStorage.multiRemove(['authToken', 'userData', 'userType']);
      // Redirect to login
    }
    return Promise.reject(error);
  }
);
```
